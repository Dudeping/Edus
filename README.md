这是一个我们小组在实训中的一个小作品，下来完善了一下。

主要实现以下功能：
简单XSS防护
邮件找回密码(腾讯企业邮)
异步搜索、异步添加、异步删除、异步修改、异步分页
批量导入(异步上传文件)
文本编辑器
分级权限控制

所用技术
BootStrap + JQuery
ASP.NET MVC + ASP.NET Identity + Entity Framework

使用软件
Visual Studio Community 2015 + MSSQL 2014 Express

部署环境
Windows Server 2012R2 + IIS
